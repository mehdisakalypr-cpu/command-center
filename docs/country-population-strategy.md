## 🛡 Principes fondateurs (non négociables)

1. **Souveraineté actionnariale** : le founder ne cède **JAMAIS plus de 20 % du capital cumulé** sur la durée de vie de l'entreprise.
2. **Zéro IPO** : pas de sortie publique. Liquidité via dividendes sur free cash flow + secondary sales ponctuels.
3. **Term sheet imposé** : le founder dicte les termes, pas les investisseurs. Drag-along asymétrique, preferred 1x non-participating, no anti-dilution, ESOP contrôlé.
4. **Investisseurs choisis** : family offices africains / sovereign wealth funds / strategic corporates minoritaires. **Jamais de fonds activistes ou pressure-IPO.**
5. **Croissance externe via M&A** autorisée, jamais par croissance VC.
6. **Data native + crowdsourcée** = moat absolu. Les entrepreneurs contribuent leurs volumes → data que Bloomberg/McKinsey/Statista ne peuvent ni acheter ni reproduire.
7. **Gouvernements = canal de distribution**, pas clients classiques. Pitch politique "souveraineté + relocalisation", pas commercial.

## 🎯 Le game-changer : double network effect natif

### Côté offre (producteurs / entrepreneurs)
- Exposent leurs volumes, catégories, saisonnalité, coûts, capacités export
- **Intérêt** : accès à des acheteurs mondiaux qui ne les trouvaient pas avant
- **Volume critique** : dès 1 000 producteurs par pays, la data devient suffisamment fine pour supplanter les estimations FAO/World Bank sur le pays

### Côté demande (acheteurs internationaux)
- Filtrent par pays, produit, volume, qualité, certifications, langue
- Contactent directement les producteurs via la plateforme
- **Intérêt** : désintermédiation (pas de trader intermédiaire prenant 15-30 %)

### Côté gouvernement (3e acteur)
- Pousse ses citoyens producteurs sur la plateforme pour souveraineté économique
- Branche ses données douanières officielles en échange d'une copie enrichie (mutualisée)
- **Intérêt** : augmenter le commerce extérieur de son pays sans dépendre de consulting étranger

### Effet combiné
- Plus d'utilisateurs → plus de data → plus précis → plus attractif → plus d'utilisateurs
- **Aucun concurrent privé ne peut copier ça** parce qu'il faut d'abord avoir la masse pour attirer les contributeurs
- **Aucun concurrent étatique (Alibaba Gov, Huawei Cloud) ne peut gagner** parce que sa neutralité géopolitique est douteuse

## 🌱 Tier "Solo Producer" €19.99/mo — segment de masse post-IA

### Contexte sociologique (ajouté 2026-04-17)

L'IA déplace massivement les emplois tertiaires (30-40 % des cols blancs menacés sur 2026-2035, McKinsey GI). Cette vague crée un reflux vers :
- **Métiers manuels** (artisanat, agriculture, construction)
- **Auto-suffisance** (potagers, micro-fermes, permaculture, homesteading)
- **Acquisition d'un lopin de terre** par peur de perdre son emploi (400k néo-paysans projetés France d'ici 2030 — INRAE)
- **Circuits courts local-to-local** (AMAP, CSA, marchés fermiers ×3 en 5 ans)

**Marché adressable** :
- France : 2-3M personnes envisagent installation rurale/micro-agri d'ici 2030
- Europe : 15-25M
- USA + Canada : 30-50M homesteaders + aspirants
- **Monde** : 200-400M (inclut Afrique rurale, Asie du Sud, LatAm)

### Le produit

**Tunnel serré, prix accessible** : **€19.99/mo** pour 1 pays × 1 opportunité × parcours complet.

**Inclus :**
- Fiche opportunité complète (volumes, marges, concurrents locaux)
- Business plan micro-production (PDF + Excel financier)
- 10 modules formation vidéo (plantation / transformation / packaging / vente)
- Matching fournisseurs locaux (semences, matériel, emballage, transport)
- Boutique en ligne intégrée (`{prénom}.feel-the-gap.shop`)
- Matching clients B2C locaux (AMAP, bio, restaurants, marchés)
- Assistant IA 24/7

**Add-ons (upsell naturel) :**
- +1 opportunité : +€5/mo
- +1 pays : +€5/mo
- Certification bio / HVE : €99 one-shot
- Coaching humain mensuel (30 min visio) : +€29/mo
- Accès financement (dossier complet + intermédiation BPI/région) : 2 % origination fee

### Pourquoi ça amplifie la souveraineté

1. **États adorent** : solution pour chômeurs tertiaires + souveraineté alimentaire = double argument politique
2. **Moat data agricole hyperlocal** : 1M fermes géolocalisées avec rendement, méthode, climat
3. **Marketplace take-rate direct** : producteur → consommateur B2C via FTG = 3-5 % commission
4. **Partenariats institutionnels** : ADEME, INRAE, Terre de Liens (FR), FAO, IFAD, AGRA (Afrique), NERPIS (Asie)
5. **Marketing zéro coût** : contenu YouTube/TikTok permaculture-homesteading (~50M audience EN/FR)

| Fraud / fake signups | Email verif + anti-sybil IP + KYC soft au 1er add-on |

## 🛠 Stratégie capital détaillée

### Phase 1 — Bootstrap (2026-2027)
- **Self-funded** via ARR early + top-up packs + tier Ultimate (299 €)
- **Aucun investisseur externe** sauf family members / angels sans droit de vote
- **Cible** : €50M ARR fin 2027

### Phase 2 — Round unique stratégique (2028)
- **Montant levé** : €20-30M maximum
- **Dilution cible** : 10-15 % (on ne redescend jamais sous 85 % founder)
- **Types d'investisseurs** (par ordre de préférence) :
  1. **Family offices africains** — Dangote (NGA), Sawiris (EGY), Otedola (NGA), Saad bin Laden group (MAR)
  2. **Sovereign wealth funds** — Mubadala (UAE), PIF (KSA), Temasek (SGP), CDG (MAR)
  3. **Strategic corporates minoritaires** — Orange (accès Afrique francophone), Safran, Bpifrance (influence politique FR)
  4. **Fonds impact développement** — BIO (BE), Proparco (FR), AfDB equity
- **Termes non négociables** :
  - Preferred 1x **non-participating** (pas de double-dip)
  - **No anti-dilution** (ratchet interdit)
  - **Drag-along asymétrique** : le founder peut forcer vente, pas les investisseurs
  - **Pay-to-play** sur rounds suivants (s'ils ne suivent pas, ils reconvertissent en common)
  - **Board** : 5 sièges dont 3 founder/ESOP + 1 investisseur + 1 indépendant. Veto founder sur toute cession de contrôle
  - **No IPO clause** : accord écrit que la société ne fera pas d'IPO sans accord unanime du founder
  - **Dividends allowed** dès Y3 (pas de réinvestissement forcé du FCF)

### Phase 3 — M&A croissance externe (2029-2030)
- **Cash + earnouts**, pas d'equity pour les vendeurs (éviter dilution)
- **Cibles priorités** :
  - **Trading Economics** (~$150M valorisation) — remplacer par cash + minority equity dans FTG
  - **IHS Maritime subsets** — licensing + asset purchase selected
  - **Boutiques consulting trade** (5-10 deals à €2-10M chacune) — achat au multiple 1-1.5× CA
  - **Plateformes locales similaires** (alibaba local clones en Afrique) — acquihire + shutdown
- **Budget total M&A** : €80-150M cumulé sur Y3-Y4 (financé via FCF + seconde tranche round unique ou mezzanine dette)

### Phase 4 — Dividendes + liquidité (2032+)
- Dès FCF positif net et stable, **distribution dividendes** :
  - 30-50 % du FCF en dividendes
  - Reste en investissement organique + M&A opportuniste
- **Secondary sales** ponctuels si besoin liquidité (jamais > 5 % du cap en une fois)
- **Zéro IPO, zéro vente majoritaire**

## ⚠️ Risques spécifiques à cette stratégie

### R1 — Adoption gouvernementale lente
- **Scénario** : < 10 states signés en Y2, modèle cale
- **Mitigation** :
  - Attaquer 6 pays pilotes alignés dès M1
  - Proposer **pilot gratuit 12 mois** pour les 3 premiers (Sénégal/Côte d'Ivoire/Maroc)
  - Communiquer massivement première signature pour déclencher FOMO voisins

### R2 — Concurrence géopolitique (Chine / US)
- **Scénario** : Alibaba Gov Cloud ou AWS GovCloud proposent équivalent subventionné
- **Mitigation** :
  - Positionnement **neutralité géopolitique** explicite (ni US ni Chine, basé en Europe)
  - Alliance stratégique avec l'UE (European Digital Sovereignty narrative)
  - Hébergement multi-cloud par pays (OVH, Scaleway, cloud local africain genre Africa Data Centers)

### R3 — Régulation data par pays
- **Scénario** : chaque pays veut localiser les données de ses citoyens (RGPD-like)
- **Mitigation** :
  - **Architecture multi-tenant par pays** avec hébergement local dès le MoU
  - Certifications locales (ISO 27001 local, HDS FR, etc.)
  - Juridique dédié par région (Afrique francophone, Afrique anglophone, Asie, LatAm)

### R4 — Monétisation marketplace lente
- **Scénario** : producteurs informels, cash-in-hand, difficulté à capter les transactions
- **Mitigation** :
  - **Intégration mobile money native** (déjà prévue — memory `reference_mobile_money_africa`)
  - Partenariats banques locales pour escrow
  - **Incentive producteur** : premier achat gratuit (pas de take-rate) pour amorcer le reflex "je paie via FTG"

### R5 — Dilution rampante via M&A
- **Scénario** : les vendeurs cibles refusent cash et demandent equity
- **Mitigation** :
  - **Cash + earnouts** uniquement, aucune equity (sauf minority 1-3 % max par deal sur les très gros)
  - Réserve mezzanine dette à disposition (€50-100M facility)
  - Abandonner les deals où les vendeurs demandent > 5 % equity

### R6 — Burn-out founder (seul aux commandes)
- **Scénario** : croissance trop rapide, impossibilité de déléguer
- **Mitigation** :
  - Recrutement **COO opérations** dès €50M ARR (Y1-Y2)
  - **CFO senior** dès le round unique (Y2)
  - **VP Government Affairs** dédié (Y2-Y3), profil ancien diplomate ou ancien ministre
  - ESOP **généreux** pour attirer top talents sans trop diluer founder (cap à 8-10 % cumulé)

1. **Le moat n'est PAS la tech — c'est le réseau**. La tech se copie, le réseau pas.
2. **Les états sont des canaux, pas des clients** : ils nous amènent les utilisateurs, pas le revenue principal.

5. **Dilution < 20 % = liberté stratégique** : aucun investisseur ne peut nous forcer à pivoter ou à vendre.
6. **Narrative "souveraineté du Sud"** est un moat géopolitique + marketing à zéro coût.
