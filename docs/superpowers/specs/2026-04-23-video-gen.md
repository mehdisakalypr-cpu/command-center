## Vision
Brief (1-3 phrases) → vidéo explainer auto 30-90s, voix multi-langues (40 langs via ElevenLabs), exports 480p/720p/1080p/16:9/9:16/1:1. Cible: solopreneurs, SaaS founders, marketers — "Loom for AI-generated explainers".

## Stack technique

**Providers IA**:
- **Seedance 2** (Enhancor provider, $0.40/s — défaut, cf. `reference_seedance_providers.md`) → génération scènes T2V
- **ElevenLabs TTS** (40 langues, ~$0.15/1000 chars) → voix off
- **ffmpeg** (assembly serveur, libx264 + AAC, crossfade audio + scene transitions)

**Briques réutilisées** (Kakashi Copy Ninja):
- `lib/ai-pool` — scénariste LLM (OpenRouter + Groq fallback, cascade coût)
- `lib/auth-v2` — sessions Supabase partagées (site_access)
- `lib/geo-pricing` — 195 multiplicateurs PPP par pays
- `lib/i18n` — UI 40 langs, cohérence absolue contenu user-facing

**Queue pattern**: BullMQ-lite via polling table `video_jobs` (status='pending') — 1 cron Vercel toutes 30s picks N jobs max → pas de Redis externe, zéro FIXED/HYBRID coût.

## COGS & marge

Vidéo 60s 1080p typique:
- Seedance 2: 60s × $0.40 = $24 ... FALSE, recalcul: cost = 1 clip 60s ≈ $2.40 (Seedance facture par clip 5-10s, 6-12 clips × $0.20-0.40)
- ElevenLabs: ~120 mots × 6 chars = 720 chars × $0.00015 = $0.11
- ffmpeg compute (Vercel Function 2min): ~$0.02
- Storage/CDN: ~$0.05
- **Total COGS: ~$3.60/vidéo**

Marge brute par tier:
- Pro €49: 20 vidéos × $3.60 = $72 → marge €49 - €66 = **NÉGATIF** ⚠️ → soit baisser volume (10 vidéos) soit défaut 720p 30s (COGS ~$1.80) → marge 93%
- Team €199: 100 vidéos 1080p 30s ≈ $180 → marge 10% (trop serré) → cap 100 vidéos 30s max + hard limit

**Révision tiers recommandée** post-checkpoint: Pro 10 vidéos / Team 50 vidéos.

## Architecture

```
Client brief → /api/video-gen/jobs (POST)
  → insert video_jobs (status=pending, tier check)
  → cron /api/cron/video-worker (30s polling)
    → LLM scénariste (ai-pool) → scenes_json (5-12 scènes)
    → pour chaque scène: Seedance prompt → media_url
    → ElevenLabs TTS par scène → voiceover_url
    → ffmpeg assembly → output_url (Supabase Storage)
    → status=completed
  → user notif email + dashboard /app/videos
```
